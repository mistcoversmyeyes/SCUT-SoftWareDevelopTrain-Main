# Iterations Index

本目录记录按周推进的阶段式迭代，不是敏捷 sprint 目录。

命名建议：

```text
week1-login-tabs.md
week2-<topic>.md
week3-<topic>.md
```

每个周迭代文件应包含：

- 背景输入
- 本周目标
- 工作包列表
- 明确不做的范围
- 验收标准
- 验证记录
- 遗留技术债与产品债链接
