# Specs Index

本目录存放需求、设计和工作包规格。规格先描述行为和验收，再指导实现。

建议分类：

- `version-*.md`：阶段需求基线。
- `wp-*.md`：具体工作包规格。
- `module-*.md`：入库、出库、库存、条码、基础信息、用户权限等模块规格。
- `integration-*.md`：MES、ERP、SAP、条码设备、手持终端等外部边界规格。

写规格时必须区分：

- 已由产品资料或课程要求确认的事实。
- 从当前代码推导出的实现现状。
- 仍待确认的产品语义。
